# CS532

Coursework repository for CS532.

## Week 01

The Week 01 work is in `CS532/W01`.

### Files

- `hello.c`: prints `Hello World!`
- `lab1.c`: checks whether an entered integer is prime

### Compile and run

From the repository root:

```sh
gcc -Wall -Wextra -std=c11 -o CS532/W01/hello CS532/W01/hello.c
./CS532/W01/hello

gcc -Wall -Wextra -std=c11 -o CS532/W01/lab1 CS532/W01/lab1.c
./CS532/W01/lab1
```
